Lesson 1: HTML

I) What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating web pages
  • HTML describes the structure of a web page
  • HTML consists of a series of elements
  • HTML elements help the browser understand and format the content
  • HTML elements label pieces of content such as “this is a title”, “this is a paragraph”, “this is a link”, etc.

II) Structure of an HTML5 document

  • <!DOCTYPE html> specifies that this document is an HTML5 document
  • <html> is the root element and contains the entire document
  • <head> contains general information (title, encoding, etc.)
  • <html lang="fr">: specifies the language of the content
  • <meta charset="UTF-8">: defines the character encoding (UTF-8)
  • <title>: sets the page title (shown in browser tab)
  • <body>: contains all visible content (titles, paragraphs, images, tables, lists…)

III) What is an HTML element?

An HTML element is made of:

  • a start tag
  • content
  • an end tag

Example:

<tag> The content goes here... </tag>

Example with real tags:

<h1> My first heading </h1>


<p> My first paragraph. </p>
